# Receipt Processor (Lab 1)

This project extracts structured information from receipt images using a language model.

## Features
- Extracts date, amount, vendor, and category from receipt images
- Supports printing extracted data as JSON
- Includes sanity checks to normalize receipt amounts
- Documentation generated with pdoc

## Requirements
- Python 3
- OpenAI API key set as an environment variable

## Usage
Activate your virtual environment, then run:

```bash
make run
